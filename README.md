 E-Commerce Website

A full-stack E-Commerce web application built using Django that provides a seamless online shopping experience. 
The platform allows users to browse products, manage their shopping cart, place orders, and securely manage their accounts through an intuitive interface.

---

### Features

### User Management
- User Registration
- User Login & Logout
- Profile Management
- Secure Authentication

###  Product Management
- Product Catalog
- Product Categories
- Product Details Page
- Product Search & Filtering

###  Shopping Cart
- Add Products to Cart
- Update Product Quantities
- Remove Products from Cart
- Cart Total Calculation

###  Order Management
- Place Orders
- Order History
- Order Status Tracking

### Admin Panel
- Manage Products
- Manage Categories
- Manage Users
- Manage Orders
